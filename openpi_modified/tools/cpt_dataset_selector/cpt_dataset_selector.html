表 · 主要 HTTP API（节选）

方法路径作用
GET/api/health存活检查
GET/api/datasets索引中出现的 dataset 族名列表
GET/api/taxonomy原子动作 / 物体 / 场景选项与别名
POST/api/query按筛选条件返回 repos 及汇总 episodes、时长
GET/api/repo/{repo_id}单库 meta + tasks 明细（当前前端未用）
GET/api/search?q=FTS 全文检索（当前前端未用）
POST/api/exportbody: repo_ids + format
环境变量：CPT_HOST、CPT_PORT（默认 9897）、CPT_INDEX_DB、CPT_TAXONOMY。索引构建侧常用 OPENPI_ROOT_DIR 或 --root-dir。

命令速贴（仓库根目录）

uv run python -m tools.cpt_dataset_selector manifest --out tools/cpt_dataset_selector/data/manifest.json

uv run python -m tools.cpt_dataset_selector index \
  --root-dir "${OPENPI_ROOT_DIR:-/mnt}" \
  --manifest tools/cpt_dataset_selector/data/manifest.json \
  --db tools/cpt_dataset_selector/data/index.sqlite3

uv run python -m tools.cpt_dataset_selector.server
